Plan New Business Defaults

Select the default values to be used (displayed) on creating a new policy. If they represent a field that is not included on the new business capture screen, they cannot be changed. For new business created by an import process, this is the way field values can be set.

Please note that when creating a new plan by using the clone plan button on product workbench, these settings are not copied.

Settings are stored in a file called LFDEFAULTS. When copying a plans settings from one data environment to another (e.g. test to live), remember to copy the LFDEFAULTS record as well as the PLANS record.

Income Option

Select the default regular income payment option from the dropdown list

Death Benefit Option

Select the default death benefit option from the dropdown list.

Options are:

         0 - Fixed Amount

         1 - Percent of Total Premium (with stopped indexation)

         2 - Percent of Premiums Paid

         3 - Face Amount + Topups - Withdrawals

         4 - Percent of Policy Value

         5 - Policy Value for 3 years then % of total premium

         6 - Straight line reducing

         7 - Percentage reducing

         8 - Use Base product Sum Assured

         9 - Straight line reducing to zero over term

         10 - Straight line increasing

         11 - Loan amount reducing (monthly)

         12 - Straight line increasing percentage

         13 - Loan amount reducing (annual)

         14 - Fixed Amount multiplied by term left

         15 - Loan repayment amount * Annuity Term

         16 - Loan amount reducing (quarterly)

         17 - Total of monthly loan repayments left

         N - No death cover
