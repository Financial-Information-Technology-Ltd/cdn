If unticked, a separate statement will be produced for each branch under which the agent has received commission.

e.g. SLA target processing time is 3 hours

Credit Card Tokenization

Tick if credit card tokenization is used. This is where actual credit card details (number, expiry date) are not stored in LIFEfit itself; they are stored by a third party, which allocates a token to represent the credit card which is stored in LIFEfit .

LIFEfit makes a webservice call to the third party on creating a new credit card, and receives a token. Thereafter, when requesting payment, the token is passed to the third party who make the actual request.

Functionality was developed as part of PCI compliance see http://en.wikipedia.org/wiki/Tokenization_%28data_security%29 for further details.
