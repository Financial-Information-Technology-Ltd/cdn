Pseudo Code Mapping

Displays all the general ledger account number mappings for an account pseudo. They can be updated.

Label

Description

Pseudo code

Accounting pseudo code, as selected on the previous screen. Display only.

Description

Accounting pseudo code description, as set at Support > Accounting > Pseudo Codes. Display only.

Account display / entry section

Company

List of companies defined at Support > Organisation > Company

Shown in all cases.

For plan or fund level accounts, if the pseudo is not mapped at plan or fund level as appropriate, a posting will be made to the company-level account.

Plan

Shown for plan-level account pseudos

Lists plan codes funds defined at Support Product Workbench

Enables different general ledger accounts to be mapped to the pseudo for different plans, giving maximum granularity (e.g. ability to analyse premium by plan)

Fund

Shown for fund-level account pseudos

Lists unit-linked funds defined at Support > Codes > Fund Codes

Enables different general ledger accounts to be mapped to the pseudo for different funds, giving maximum granularity. (e.g. ability to analyse administration charge by fund)
