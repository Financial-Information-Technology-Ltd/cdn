Currency Codes

Set the parameters of a currency to be used in LIFEfit

Label

Description

Currency Code

ISO4217 three character currency code as selected or entered on the previous screen

Currency name

Name of currency defaults to the ISO4217 currency name but can be changed

e.g. you may wish to call GBP British Pounds instead of the default Pound Sterling.

No of decimals

Number of decimal places used in the currency

e.g. Pound sterling, US Dollar and EURO all have 2 (pence or cents). Zambian Kwacha has 0. Bahraini Dinar has 3.

Active

Tick the checkbox if the currency is to be used in LIFEfit. This will enable its selection when using LIFEfit multi-currency functionality
