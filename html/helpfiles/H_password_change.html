Password Change

This screen is used to change or setup new user password.

Label

Description

Operator

User ID (used when logging in to LIFEfit )

Name

User Name

Your Password

The password is hidden.

As a security measure when changing password you will have to enter your own LIFEfit password first.

New Password

This is the new password you will enter. Again this password is hidden.

Retype New Password

For security measures the new password needs to be typed in again.

Force password change at next login

This field is a check box.

When checked in LIFEfit will force user to change password at next login

Password Never Expires

This field is a check box.

Password on LIFEfit will never expire once this box is checked in.
