Choosing account

Select the nominal ledger (general ledger) account number whose features you wish to edit. This will define what is output in the nominal ledger interface file, produced in batch to feed the corporate general ledger application e.g. Sun Accounts.

Label

Description

Account No

Enter the general ledger account number, or enter a partial account number and click <Search Account>.

<Search Account> button

Clicking this button returns a list of all the account numbers with a partial match, as entered in Account No. You can select the account you wish to edit by clicking it in the list.

Branch

Select the branch code from the dropdown list.

Currency

Select the currency from the dropdown list.

<Continue> button

Continue to edit the account numbers parameters.

<Return> hyperlink

Return to the main menu.
