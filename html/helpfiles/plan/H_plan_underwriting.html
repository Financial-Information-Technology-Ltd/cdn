Plan Underwriting

Underwriting rules that will be applied to policies at new business and reinstatement.

Underwriting codes can either be set up individually for each plan, or in a table which can be used by multiple plans

Label

Description

Plan Code

Plan Code display only

Plan Name

Plan Name display only

Code

Select an underwriting code from the dropdown list.

The list is comprised of underwriting codes defined at Support > Underwriting > Requirements. These are typically medical tests (the higher the sum assured or older the life assured, the more tests are required) or know your customer type procedural checks.

Basis

Select the underwriting basis from the dropdown list.

Options are:

         ALW - Always

         ALWS - Always for Base Policy (not segments)

         DOC_COMP - Issue documents complete

         FACE - Face amount

         FCL - Sum Assured Above FCL

         FCL_MEM - Percentage of members exceeding FCL

         GRP_MEM - Number of members in a group (group business)

         GRP_M_PLAN - Number of members in a group with this plan

         HTWT - Height/Weight (BMI) check

         LIVES - Any life assured

         NON_MED - Non-medical underwriting required

         OWNERS_IND - Individual owners

         OWNERS_COMP - Company Owners

         OWNERS_AGE - Check Owners age

         TFAL - Total face Amount (Life Assured)

         TPPL - Total Premiums Payable (Life Assured)

         TPPO - Total Premiums Payable (Owner)

Minimum Value

For bases which require an underwriting code if the value falls within a range (e.g. owners age), enter the minimum value here.

Maximum Value

For bases which require an underwriting code if the value falls within a range (e.g. owners age), enter the maximum value here.

Created at New Business

Tick if the underwriting check is applied at new business. In most cases this will be required.

Created at Reinstatement

Tick if the underwriting check is applied on reinstatement form lapse or paid-up.

Create for riders also

Tick if the underwriting requirement should be applied to rider coverages (as well as for the base)

Terminate Declined Coverages

Tick if a coverage should be terminated if the underwriting requirement is declined.

<Add> button

Add another underwriting code to the list

Underwriting table is currency-specific

Tick if the underwriting table is currency specific.

If ticked, you will be presented with a currency-specific list of underwriting tables

Underwriting table

Select an underwriting table from the dropdown list.

The list is comprised of underwriting tables defined at Support > Underwriting > Table

Underwriting table is country-specific

Tick if the underwriting table is country specific.

You will be shown a list of countries in which the plan is available, and be able to select an underwriting table form a dropdown for each of them.
