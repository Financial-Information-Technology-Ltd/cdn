Plan Sum Assured Details

Sum Assured settings for the plan

Label

Description

Minimum Sum Assured

The minimum Sum Assured allowed for this plan.

Maximum Sum Assured

The maximum Sum Assured allowed for this plan.

Maximum Sum Assured Calculation

If a Maximum Sum Assured value is set, this parameter defines how it is applied to the policy. The available options are:

0 - Current Policy against Max on Plan

1 Check all policies against Life Assured for Max on Plan

2 Pre 1996 Aberdeen formula

3 Post 1996 Aberdeen formula

Sum Assured Options

Defines how the Sum Assured (Benefit Amount) is calculated. Select from the dropdown list. Options are

         Fixed Amount

         Percent of Total Premium (with stopped indexation)

         Percent of Premiums Paid

         Face Amount + Topups - Withdrawals

         Percent of Policy Value

         Policy Value for 3 years then % of total premium

         Straight line reducing

         Percentage reducing

         Use Base product Sum Assured

         Straight line reducing to zero over term

         Straight line increasing

         Loan amount reducing (monthly)

         Straight line increasing percentage

         Loan amount reducing (annual)

         Fixed Amount multiplied by term left

         Loan repayment amount * Annuity Term

         Loan amount reducing (quarterly)

         Total of monthly loan repayments left

         No death cover

Allowed Sum Assured percentages

Can be used where sum assured is a percentage of policy value

If there is a fixed set of sum assured percentages that are offered, enter the valid percentages here.

If this feature is used, the user will be presented with a dropdown list of these values at quotation or new business entry

Allowed Sum Assured

If there is a fixed set of sums assured that are offered, enter the valid sums assured here.

If this feature is used, the user will be presented with a dropdown list of these values at quotation or new business entry

Minimum Sum Assured Calculation percentage

Minimum sum assured as a percentage of policy value.

Sum assured percentage entered at new business will be validated against this value

Maximum Sum Assured Calculation percentage

Maximum sum assured as a percentage of policy value.

Sum assured percentage entered at new business will be validated against this value

Rider sum assured should not exceed sum assured on base coverage

Tick if this plan is used as a rider, and its sum assured is limited to no more than the sum assured of the base plan

Decrease Sum Assured automatically when above maximum

Tick if the sum assured should be automatically reduced if it exceeds the maximum permitted.
