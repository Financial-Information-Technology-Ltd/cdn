Plan Withdrawals

LIFEfit supports partial withdrawals from both unit-linked and deposit administration (interest bearing) savings product types.

If possible, the withdrawal is processed online. If not, because a unit price or exchange rate is required and not yet available, a FUWD future record is created and processed by batch job LFS.FUWD.

Allow MVA change on surrenders and withdrawals

An MVA (Market Value Adjustment) can be charged on cancelling units in with profits funds, to represent the difference between the underlying asset value of the fund and its stated unit price.

If an MVA is calculated for partial withdrawals and surrenders, this allows the calculated MVA to be overridden on screen.

Maximum Free Withdrawal Details

Maximum Free Withdrawal Basis

If some free (i.e. without penalty) partial withdrawal(s) is/are permitted, select the basis here from the dropdown list.

Options are:

         1 - Per withdrawal transaction (i.e. penalty is only charged on withdrawal amount above this figure)

         2 - Per policy year (a counter is maintained, and reset on policy anniversary, to determine whether to apply a penalty)

Maximum Free Withdrawal Calculation Code

Select the calculation code for maximum free withdrawal from the dropdown list.

Options are:

         1 - Flat Amount

         2 - % of Value of Fund(s)

         3 - % of Original Investment

Allowed Withdrawal Types (if empty - all relevant allowed)

Select the permitted partial withdrawal types from the dropdown list.

Options are:

         N - Net amount

         F - Net amount (fund based)

         U - No. of Units (fund based)

         V - Value of Units (fund based)

         S - Percent of Value

         P - Percentage of Units (fund based)

The user will select from a dropdown list of permitted partial withdrawal types when requesting the withdrawal.

Allowed Termination Reasons

Select the allowed termination reasons from the dropdown list. This is usually used for pension policies which have regulatory restrictions on withdrawals.

Options are:

         FSUR - Full Surrender

         SSUR - Segment surrender

         PSIL - Pension Withdrawal, Ill Health Retirement

         PSES - Pension Withdrawal, Emigration Settlement

         PSLR - Pension Withdrawal, Late Retirement Settlement

         PTRF - Pension Withdrawal, Transfer to another provider

         PERE - Pension Withdrawal, Early Retirement

         PNRE - Pension Withdrawal, Normal Retirement

The user will select from a dropdown list of permitted termination reasons when requesting the withdrawal.
