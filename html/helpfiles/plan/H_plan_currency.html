Plan Currency Specific Amounts

Plans can be administered in multiple currencies. This screen is used to set plan parameters in different currencies all parameters shown here are set in the relevant product workbench screen for the default plan currency.

This data is stored in file PLANS_CURRENCY, which should be remembered when moving plans between data environments. The key to PLANS_CURRENCY is PLANCODE*CURRENCYCODE.

e.g. TERM*USD.
