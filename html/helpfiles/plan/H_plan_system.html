Plan System Processes

Parameters defining the back office servicing screens to be used. Different sets of screens are used for different product types.

New business process

Select the back office new business entry screen to be used from the dropdown list.

If this is selected and the plan is available for new business (as defined at Product Workbench > New Business), it will be available in the dropdown in the new business screen.

New Coverage process

For riders that are available post-issue, select the screen to be used when adding the rider from the dropdown list.

New Business Coverage agent process

For riders that are available post-issue, select the screen to be used when entering agent details as part of adding the rider from the dropdown list.

New Business Policy updated screen

Select the new business policy updated screen from the dropdown list.

Fund Allocation process

Select the screen to be used to select unit-linked investment funds as part of new business entry (or redirection) from the dropdown list.

Select the policy maintenance screen from the dropdown list.

Select the beneficiary maintenance screen from the dropdown list.
