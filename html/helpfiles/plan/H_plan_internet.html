Do not edit application if created via quotation functionality

Tick if data entered in a pre-sales quotation cannot be altered in the application screen. Every application must start as a quotation.

Protect funds on application after quote

Tick if funds selected on the quotation cannot be changed in the application. This is typically required where assumed growth rates in a pre-sales projection of maturity value are fund-specific. It ensures that any post-sales projection, which may be required for regulatory compliance, will match.
