Plan Setup also known as Product Workbench

In LIFEfit, insurance products are built from a set of one or more plan codes. Plan codes equate to coverages on a policy.

This screen provides hyperlinks to screens on which all plan parameters are set. It also allows you to create new plans by cloning (copying) existing ones.

Products can be moved from one environment to another (e.g. from test to production) using the Datamove tool. Please note you will probably need to include data in tables other than PLANS. For example, data from the following tables may also be required:

       COMMRATES (commission rates)

       COMMTYPES (commission types)

       DOCUMENT_MERGE (document setup)

       EVENT_HANDLER (document configuration)

       INTRATES (interest rates for deposit administration and loans)

       LFDEFAULTS (plan default values)

       PLANS_CURRENCY (currency-specific values)

       SCHEME_RATES (rate structure definition)

  • UNIT_VALUES (rates)
