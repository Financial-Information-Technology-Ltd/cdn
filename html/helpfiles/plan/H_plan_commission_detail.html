Plan Commission Detail

Parameters defining the Agents Commission set-up for this Plan

Label

Description

Plan Code

Plan Code display only

Plan Name

Plan Name display only

Commission Type

Select the commission type from the dropdown list.

The list is comprised of commission types defined at Support > Agent > Commission Types.

Fixed Commission Amount

For commission types where a fixed amount of commission is paid, irrespective of the premiums, enter the amount here.

Rider Commission

For plans used as a rider, select from the dropdown list to determine whether they have their own commission type, or use that of their base plan.

Override Commission Calculation

Select the override commission calculation type from the dropdown list.

Options are:

         0 - Use override agent type to look up base commission rates

         1 - Apply override percent to writing Agent's commission

         2 - Same as 1, but checks if table is set on plan

         3 - Apply override percent to net premium

Override Percent Table (to be used with calculate code "2")

Used in conjunction with Override Commission Calculation 2 above. Select the table from the dropdown list.

The list is comprised of tables defined at Support > Rates > Scheme Rates.

How many years to pay override renewal commissions

Enter the number of years for which override renewal premiums should be paid.

If blank, they will be paid until policy termination.

Prefinanced rate table

Select the prefinanced rate table from the dropdown list.

The list is comprised of tables defined at Support > Rates > Scheme Rates.

Use same agent as on main plan

If this plan is a rider, tick if it should use the same agent as the base plan.

Allow Pattern Code Override

Tick if a different commission pattern code should be used from the one defined in the commission type selected above.

Pattern codes

Select the override pattern code to use form the dropdown list.

The list is comprised of codes defined at Support > Agent > Commission Rates.

Commissions By Agent Type

Allows fixed Commission Rates to be set for this Plan. These rates will be used for the writing and override agents, instead of the usual commission rates set at Support > Agent > Commission Rates. If Renewal Override Percentage is blank for the agent type it will check the renewal override for the agent.

Consultant type

Select the consultant type from the dropdown list.

Clawback Basis

Unearned indemnified (up front) commission can be clawed back from the agent if the policy is terminated.

Select commission clawback (on policy termination) basis from the dropdown list.

Options are:

         0 - Unearned balance

         1 - Proportion of the commissions paid since clawback period ago

         2 - Proportion of the commissions paid since issue

         3 Proportion of the commissions paid since issue, varying by term. Formula is [1-(months elapsed at surrender/months in policy term)] * [clawback factor] * [commission paid]

Clawback Period (months)

Used in conjunction with Clawback Basis 2 above.

Enter the clawback period in months.

Clawback Rate Table

Select the clawback rate table, to be used to set the proportion in Clawback bases1 and 2 above, from the dropdown list.

The list is comprised of tables defined at Support > Rates > Scheme Rates.

Accounting when writing off commissions

Tick if commission write-off should be accounted for as bad debt.

Init Comm Give Up Code

Select the initial commission give-up (sacrifice) code from the dropdown list.

Options are:

         0 - Not Applicable

         1 - Adjustment to Investment Amount

         2 - 1:.8 Adjustment to Investment

         3 - No effect on Investment Amounts

         4 - 1:.8 Adjustment to IPC

Max Init Comm Give Up

Enter the maximum initial commission give-up (sacrifice) amount

Earnings Frequency

For single premium commission which is earned over time, enter the commission earning frequency. This is triggered by a SPE future record which is processed by batch job LF.SINGLE.PREMIUM.EARNINGS

Commissions at status J

Tick if commission is to be paid at policy status change to J active awaiting first premium by direct debit

Commissions at complete pending status

Tick if commission is to be paid at policy status change complete pending (i.e. no outstanding underwriting requirements remain).

Complete pending status is defined at Support > Codes > Status Codes.

Save changes.
