Plan Endorsements

Screen is used to determine what types of endorsement (post-issue, client-requested policy changes) are permitted for policies of this plan type, and other rules for policy behaviour on endorsement.

Substitution plans

If the policys plan code can be changed, select the possible new plan code(s) here.

This is sometimes used when a change in regulatory environment (e.g. the policy owner moves country) requires a change in the investment funds allowed under the policy.

Conditional Special Clauses

Permitted special clauses for the product. Will be selected at new business and added to the policy schedule document.

Mandatory Special Clauses

Mandatory special clauses for the product. Will be added to the policy schedule document.

Postpone DD and lapse on premium endorsement (days)

Number of days which direct debit and lapse processing should be delayed in the event of a premium endorsement.

Activate DD on premium endorsement

Make direct debit active when premium is endorsed

Stop indexation on premium decrease

Tick this box if planned premium indexation stops when the client chooses to reduce his/her premium

Increment endorsement number on withdrawals and switches

LIFEfit maintains an endorsement count for each policy. If withdrawals and switches are to be included in the count, tick this option

When endorsing sum assured, premium is calculated outside the system and changed manually

Tick if premium is recalculated manually on endorsement of sum assured

Number of Premium Endorsements allowed.

If there is a restriction on the number of premium endorsements permitted, enter the number here.
