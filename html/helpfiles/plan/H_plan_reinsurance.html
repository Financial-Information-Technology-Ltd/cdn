Plan Reinsurance Premium Calculation

This screen is used to set the reinsurance parameters for the plan.

Label

Description

Plan Code

Plan Code display only

Plan Name

Plan Name display only

Treaty Code

Select a reinsurance treaty code for the plan from the dropdown list.

The list is comprised of reinsurance treaties defined at Support > Reinsurance > Reinsurance Treaty.

On policy issue, a REINS future record is created for each coverage with one or more treaties defined here. The REINS future record is processed by batch job LF.CALC.REINSURANCE. The date (timing) of the REINS future record is determined by the settings on the treaties.

<Add> button

Add another reinsurance treaty.

Commission Share

Enter the percentage of reinsurance premium that goes to the agent - can be used where reinsurance premium is calculated at the same time as mortality for unit-linked products

Process reinsurance at the same time as mortality (unit-linked products)

Tick if reinsurance is processed at the same time as mortality (unit-linked products) i.e. by batch job LFSUAL for that processes the UMD future record (rather than by a REINS future record)

Process reinsurance at dealing day (unit-linked products)

Tick if reinsurance is processed at the dealing day (for unit-linked environments where funds are priced less than daily frequency - typically monthly). The REINS future record will be set for the next dealing days after its due date.

Reinsurance Finance Company

Select the reinsurance finance company from the dropdown list.

The list is comprised of reinsurers defined at Support > Reinsurance > Reinsurance Company

Reinsurance Finance Rate

Enter the percentage of commission which is financed by the reinsurer.

Reinsurance Repayment Rate

Enter the percentage of fees collected from the policyholder which are repaid to reinsurer.
