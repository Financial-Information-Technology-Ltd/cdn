Plan Surrenders

Parameters for policy surrender.

Savings policies can be surrendered from the policy workbench. These parameters determine what value will be paid back to the policyholder, including any fees, charges and penalties that may be applied.

Label

Description

Plan Code

Plan code display only

Plan Name

Plan name display only

Months to surrender value

Radio button Value or Table

Select whether there is a fixed number of months before a policy can be surrendered, or if the number of months is read from a table where it is policy term dependent.

Months to surrender

If value is selected in Months to surrender value above, enter the number of months a policy must be in force before it can be surrendered.

Term calculation for surrender months

If Table is selected in Months to surrender value above, select the term calculation from the dropdown list.

Options are:

         M - Maturity or Expiry Term

         C - Contribution (Premium Payment) Term

Surrender Table

Enter the term (in years) and months until surrender is permitted for that term

e.g.

Term Months to Surrender

5 12

10 24

20 36

50 48

Surrender Calculation Code

Select the surrender value calculation code from the dropdown list.

Options are:

         0 - Surrender not available for this contract

         1 - Coverage Value * (100 - MVA percent)

         5 - Premiums Paid * Surrender Value Rate (%)

         6 - Premiums Paid * Surrender Value Rate (%) + Accrued Interest

         7 - Coverage Value - Accrued Interest * Factor

         8 - Sum Assured * Surrender Factor from a rate table

         10 Bid Value of Accumulation Units*[100-Cash.Factor]

         11 - Bid Value of Accumulation Units

         16 - Bid Value of Accumulation Units / Cash Value Factor

         17 - Bid Value of Accumulation Units - Outstanding Establishment Charge

         19 - (Value * Surrender Rate) * (100% - Surrender Factor)

         20 - Coverage Value - Total Premium*Penalty Factor

         22 - Bid Value of Company Units(Initial + Accumulation)

         23 - Bid Value of Company Units(Initial +Accumulation ) Oustanding Reduction Fee Factor Z

         24 - (Value of Initial Units * Surrender Penalty Rate) + Value of Accumulation Units

         25 - SA-(less paid-to date and inforce months/term in months)*(less paid-to date and inforce months/term in years)* Sum Assured)

         26 - Premiums Paid * Rate - Outstanding Premiums

         27 - (Pro Rata By Duration (Sum Assured + Reversionary bonus)) * Surrender Factor)

Surrender Value Rate Table

For surrender calculation codes involving a rate table, select the rate table from the dropdown list.

The list is comprised of rate tables defined at Support > Rates > Scheme Rates

Surrender Penalty Factor Rate Table

For surrender calculation codes involving a penalty rate table, select the rate table from the dropdown list.

The list is comprised of rate tables defined at Support > Rates > Scheme Rates

Fixed Surrender Fee

Enter the fixed surrender fee (if any)

Minimum Penalty calculation Code

Used in conjunction with surrender calculation code 19

Select the basis for determining the minimum penalty fee permitted from the dropdown list

Options are

         1 - Amount of outstanding Entrance Fee. This is applicable for regular premium unit-linked products where an entrance fee is charged (a fee that is taken on a regular basis at the start of the policys life). If the policy is surrendered before the end of the entrance fee charging term, the remaining entrance fee is charged.

Minimum Penalty calculation Factor

Used in conjunction with surrender calculation code 19

Enter the factor used in determining the minimum penalty fee permitted - used in conjunction with minimum penalty calculcation code 1 above (i.e. factor to apply to outstanding entrance fee)

Penalty Factor Z

Used in conjucntion with surrender calculation code 23.

Select the rate table from the dropdown list.

The list is comprised of rate tables defined at Support > Rates > Scheme Rates.

No Surrender penalty from paid-up status

Tick is it is not permitted to surrender a policy in a paid-up status.

Surrender Grace Period (days)

Numer of days after policy issue before surrender is permitted.
