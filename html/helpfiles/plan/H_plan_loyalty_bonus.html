Plan Loyalty Bonus

Loyalty bonus settings for the plan. For the fund level parameters, defining timing and frequency of the bonus, see the fund code (Support > Codes > Fund Codes)

Label

Description

Plan Code

Plan Code display only

Plan Name

Plan Name display only

Basis

Select loyalty bonus basis from the dropdown list

Options are:

         3 - Fund-based loyalty bonus (each funds bonus is calculated separately)

         12 - 12 month average (bonus is calculated on a twelve monthly average policy value)

Rate

Enter the bonus rate to use (if a fixed amount)

Rate Table

Select the rate table to be used for setting the bonus rate from the dropdown list.

The list is comprised of tables defined at Support > Rates > Scheme Rates.

Pay Method

Select the loyalty bonus allocation method from the dropdown list.

Options are:

         ALLOCATION - Allocate at bid price

         ALLOC_OFFER - Allocate at offer price

Pay only if policy Up To Date

Tick if the bonus is only applied for policies where the premium is up to date (i.e. paid-to date >= system date). This encourages policyholders to keep paying regular premiums.
