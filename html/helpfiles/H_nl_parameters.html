NL (Nominal Ledger) Parameters

These are the parameters that govern the format of the nominal ledger interface file, containing accounting journal information exported to the corporate general ledger. This is typically done daily at the end of the accounting section of the daily batch by job LF.SUN.EXPORT. The export file format is fixed-width fields by default, but can be delimited by batch parameter (tip: use **, if you want CSV format).

Label

Description

Nl allowed

Tick if export to the nominal ledger is allowed.

Nl journal type

Enter the nominal ledger export type. This data is written to position 68 in the export file.

Nl journal source

Enter the nominal ledger journal source. This data is written to position 73 in the export file.

Nl last transfer date

Date of the most recent transfer of data to the nominal ledger (display only).

Nl deb cred

Select from the dropdown list. Options are

       C Creditors

       D - Debtors

This determines whether the export is seen from a creditors or a debtors perspective whether LIFEfit transactions are exported with the same or opposite signs as in its internal general ledger.

If you want the export to use C for credits and D for debits (the same as LIFEfits internal GL), select C Creditors.

Nl system

Select the nominal ledger system and version from the dropdown list. The export file is configured accordingly. Options are

       1 Sun 3.2

       2 Sun 3.5

       3 Sun 4.2

Please note that although the export file is configured for Sun Accounts, other ledger packages (e.g. SAS, Oracle Financials) may be configured to use data in the same format.

Policy in Description

Please tick if the policy number should be included in the description field (position 93 in the export file), in addition to the LIFEfit transaction description.

Omit Nl Header

Tick if the export file should be produced without a header row.

Nl Numeric Output Format

Select the numeric output format from the dropdown list. Options are:

0 numeric amount with leading zeros

1 numeric amount with trailing zeros

Accounting period & Nl acc period:

Enter the start date and number of each nominal ledger accounting period. The number is written to Ledger Period position 93 in the export file.

<Add> button

Add another accounting period.

<Save> button

Save changes.

<Return> hyperlink

Return to the LIFEfit main menu.
