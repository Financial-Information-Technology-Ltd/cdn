Investment Trust functionality

Label

Description

Client No

Select an investment trust from the dropdown list. The list is comprised of clients categorised as name type T.

Trust name

Enter the client number if the investment trust.

Income Payment Method to Trust

Select the income payment method from the dropdown list. Options are:

         Make one payment to trust per client

         Make one total payment for all clients

Agent Receiving Trust Fees

Agent number of the agent who receives trust fees. In order to receive trust fees, each trust has a LIFEfit agent record to enable the trust fees to be paid out through the commission payment process.

Spool directory

Directory to which documents for the trusts policies are spooled.

<Modify> button

Click to edit trust details.

<Valuation> button

Click to start the process of generating valuation statements for the trusts policies.

<Download Stored Documents> button

Click to start the process of downloading documents from DocStore for the trusts policies.

<Return> hyperlink

Return to the main menu.
