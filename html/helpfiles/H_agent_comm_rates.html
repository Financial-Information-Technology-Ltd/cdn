Commission Rates

Manage commission rates for different agent types for a pattern code. The pattern code is set on the product, and the agent type is set on the agent.

Label

Description

Commission Pattern Code

Display only - as entered (or selected) on a previous screen

Maximum rate

Display only the highest value on any of the agent types / durations on this commission rates record

Agent Type

Display only as selected on the previous screen

Duration

Duration in years

n.b. for durations greater than the highest value here, commission will be paid at the rate of the last duration shown. If you want to stop paying commission after a certain duration, you need a zero row at the end of the table for duration max duration at which commission is paid + 1.

e.g. commission is 15% in year 1, 10% year 2, 3% year 3, 0% thereafter

Duration Rate

1                                      15

2                                      10

3                                      3

4                                      0

Commission Rate

Commission rate (as a percentage of premium) for the duration

Commission Rate Table

If ticked, the commission rate is read from a rate table.

Select the rate table from the dropdown list which appears.

Commission Amount

Used if a fixed commission amount is paid, as defined on the commission type screen (Initial Commission Entitlement Code F Fixed amount)

<Add> button

Add another row to the table as applicable (for another commission duration if entering rate by duration, or another agent type if entering rate by agent type)

<Save> button

Save the commission rates record

<Return> hyperlink

Returns you to the main menu
