Run Reports

This screen is used to run reports, which can be output as CSV (for Excel), fixed-width text file or HTML (web page).

It is separated from the create reports screen some users may be permitted to run reports, but not create or amend them

Label

Description

Report name

You are presented with a list of reports to which you have access as either report owner, or as a member of a user group allowed to see the report.

Select a report name from the dropdown list.

Reports are created/amended at Support Files > Reports > Create Reports

Output format

Select your preferred output format from the dropdown list:

Options:

         .csv (for Excel)

         HTML (web page)

         .txt (fixed column width text file)

Tip 1: it can often be useful, for quick reports, to use HTML. You can click and open in new tab, and minimise the overhead of opening another application such as Excel

Tip 2: beware the loss of leading zeros in data when opening CSV files in Excel

<Run> button

Runs the report (you will be presented with a hyperlink to open the report once it has finished)

<View Selection> button

Displays the report definition (selection criteria, filter criteria and columns to be output)

<Return> hyperlink

Returns you to the main menu
