This enables the same plan codes to be sold to customers in different countries where cool-off periods vary (the length of time after policy issue during which the client can change his mind and cancel the policy without penalty)

Country Risk Category

Select from the dropdown. Options are defined in CODES record RISK_CATEGORY.

This field can be used as part of the key on a rate table, and enables country-specific rates. This is sometimes used for mortality rates.

Premium Tax Rate (percent of premium)

Percentage of premium to be accounted for as tax, on every premium received.

Accounting is

Dr COTAX

Cr PREMTAX

n.b. accounting is reversed on premium reversal

New Business Tax Rate (percent of SA)

Percentage of sum assured to be accounted for at policy issue. This is sometimes a levy paid to the national insurance regulator.

Accounting is

Dr COTAX
Cr NBTAX

n.b. accounting is reversed on policy cancellation.

How to format address in documentation

Address format for correspondence can vary according to the country of the address

Options:

         Street Address - City - County - Post Code - Country

         Street Address - County - Post Code

         Street Address - City - County

         Street Address - Post Code City - Country

Post Code Look Up Allowed

Tick if the address screens are linked to an online, third party address validation facility.

LIFEfit currently supports AFD WorldAddress which can be licensed from AFD Software (http://www.afd.co.uk/product_worldaddress.asp).

Sort Order

If linked to a third party address lookup application, this enables countries to be listed in non-alphabetical order.

Address search applications work by first selecting the country. This feature enables you to have your home country appearing first in a list which is (by default) alphabetical.

e.g.

UK

USA

Afghanistan

Albania
