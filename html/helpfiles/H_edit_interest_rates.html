Interest Rates

This screen is used to maintain interest rates used for DA (deposit administration i.e. interest-bearing) savings products. Interest is typically calculated and credited to the savings policy monthly. This is done automatically by a batch process.

This screen is also used to define interest rates to be charged on policy loans.

Label

Description

Rates Code

Interest rate code, as selected or entered on the previous screen (display only)

Rates Description

Interest rate description free format text

Rates in Table

This option is sometimes used for loan interest rate definition - tick if the interest rate varies according to loan term and loan amount

Typically, companies offer lower interest rates for longer terms and higher loan amounts

If you select this option, you will be prompted to enter the axes for the matrix (terms and amounts), as well as the rates themselves

Date

Date from which a rate is applicable

DA rates are typically declared annually, and depend on company investment returns

Rate

Interest rate to use from the applicable date

<Update> button

Updates the rate.

For DA savings, companies usually use an interim rate until the final rate is known. This ensures that policy claims can be paid out including interest capitalised to the claim date.

Once the final rate is known, it can be updated here and a process run to recalculate interest and make adjustments for the difference between the final rate and interest rate.

The interest recalculation process is run at Administration > Recalculate Interest.

<Save> button

Saves any changes

<Return> hyperlink

Returns you to the previous screen (interest rate selection)
