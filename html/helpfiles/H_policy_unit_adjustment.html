Ad Hoc Unit Adjustments

Enter details of the ad hoc unit adjustment.

Label

Description

Policy No

Policy number entered on the previous screen display only.

Coverage

If units are held on more than one coverage, select the coverage whose holdings you want to adjust from the dropdown list. Otherwise display only of the unit-holding coverage.

Unit Date

Enter the unit price date to be used for the adjustment.

Price Basis

Select the price basis from the dropdown list. Options are:

       Bid Price

       Offer Price

Unit allocation is usually done at offer price, and deallocation (e.g. for charges and risk premiums) is done at bid price.

Reason

Select the reason for the adjustment from the dropdown list. Options are:

       Unit Adjustment

       Admin Charge

       Death Benefit Payment

       Fund Loyalty Bonus Payment

       Maturity Payment

       Maintenance Fee

       Unit Deallocation for Premium

       Mortality Deduction

       Premium Payment (Premium)

       Unit Reversal

       Surrender

       Trailer Commissions

       Unit Withdrawal

       Renewal Commissions

       Agent Service Agreement Charge

       Management Fee

By selecting the reason for adjustment, you ensure the correct unit accounting is created for the transaction. The fund level and plan level margin pseudos are specific to the transaction type. In addition, the correct transaction reason is written to the unit accounting transaction record for subsequent analysis and reporting. Moreover, it is also possible to see that a transaction was created as an adjustment, rather than in regular processing.

Transaction Date

Current system date display only.

Description

Enter an explanation for the unit adjustment free format text.

Unit details section.

The funds available on the plan are listed, along with the current unit holdings on the coverage.

Enter adjustments, the number of units to be added (positive number) or removed (negative number).

<Submit> button

Process the unit adjustment (write UNITMOVES and UNITACCTRN records, and update the policy unit balance).

<Return> hyperlink

Return to the main menu.
