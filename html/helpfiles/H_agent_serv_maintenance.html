Servicing Agent Maintenance

Enter the change of servicing agent details.

The fields displayed depend on the servicing option chosen on the previous screen.

Please note that servicing agents receive renewal commission, but do not receive initial commission

Label

Description

Maintenance Option

As selected on the previous screen

New Servicing Agent

Agent number of new servicing agent

Writing Agent / Old Servicing Agent

Agent number of agent being replaced

Client Code

Client number of client whose policies are to be serviced by the new servicing agent

Policy Number

Policies that are to be serviced by the new servicing agent

Update trailer agent

Is trailer commission (unit-linked policy value-based commission) to be paid to the new servicing agent, in addition to premium-based renewal commission

<Submit> button

Save the changes to servicing agent details

<Return> hyperlink

Returns you to the previous screen (change of servicing agent option selection)
