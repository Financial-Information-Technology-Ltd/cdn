Payments Out List

This screen lists all payments out that are awaiting authorisation.

The list can be sorted by any of the headings that are "hyperlinked" - click to re-sort.

Please note that a payment can not be authorised by the user who last changed it. This is a fraud prevention measure.

Label

Description

Key

Unique identifier for the payment out record

Reason

Payment Reason (e.g. surrender, maturity, death)

Currency

Payment currency

Gross Amount

Payment amount before adjustments (the amount calculated by LIFEfit according to product rules)

Net Amount

Payment amount after adjustments (e.g. bank charges, ex-gratia payments)

Policy

Policy for which the payment has been generated

Created User

User who initiated the payment out (e.g. made the "cash surrender" status change)

Created Date

Date the payment out record was created

Created Time

Time the payment out record was created

Last Changed User

User who last changed the record. If the data has not been changed, it is the user who created it.
