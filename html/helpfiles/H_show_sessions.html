User sessions

Information about users currently logged on to LIFEfit.

n.b. login attempts (whether successful or not) are recorded in file SESSIONLOG, which can be reported from Davywriter (Support > Reports > Create Reports) or via ODBC.

Description

Login time

Time the user logged in.

Last Request

Time of most recent user activity (e.g. screen refresh, transaction created).
