Group Workbench

This screen allows you to search for a group using one or more attributes. Group Search can be narrowed down according to the group status. New groups can also be defined through this screen either by using existing client numbers or by selecting the Creating New Owner hyperlink. Another way of creating New Clients is via the Client Workbench screen ( Individual >Client Workbench) .

Label

Description

Group Client

Enter Group Scheme Owner client number. LIFEfit will list all of this clients group schemes. The Show Client button allows you to view the client record details.

New Scheme Owner

This opens up the Corporate Client entry screen. Once all the available details are entered and saved a new corporate client record is created and a unique reference code is allocated.

Group No

Each Group is identified with a unique group number. Provided the Group number exists, LIFEfit will open up the Groups Scheme Setup screen otherwise an error message will be displayed at the top of the screen.

Scheme Policy Number

Group Schemes can be identified by the Group Master Policy which is assigned to a group on sign-off or on submitting application. Once entered LIFEfit will open up the Scheme set up screen.

Plan for new group

This drop-down lists all the products available when creating a new group. The products available to the user depend on which companies and branches are approved at plan level ( Product Workbench > new Business ) and system level ( Administration > Security/Processes > Operator Profiles).

Pending groups

Generates in ascending order of the group number, a list of all groups which have an internal pending status. For example schemes in RTI (ready to issue) or UW (Underwriting).

Active Groups

Generates in ascending order of the group number, a list of all groups that have been issued (ISS/A) or reinstated (RA).

Terminated groups

Generates in ascending order of the group number, a list of all groups that have been surrendered, lapsed or terminated (SUR/L/CBA/T).

Group Names

Groups can be searched by entering the first four characters of the name of the Scheme. This can be found on the corporate client record.

Registration Number

A registration number can be used to identify the schemes. This is another unique attribute defined on the corporate client record.
