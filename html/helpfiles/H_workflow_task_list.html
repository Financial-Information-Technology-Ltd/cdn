Workflow Task List

List of workflow items, as requested on the previous screen

Heading

Description

Level

For active items, their RAG (Red Amber Green) rating, indicating their urgency. Items start out green, turn amber on their target processing date and red on their maximum processing date. Target and maximum processing days are set as an attirbute of the task type at Support > Codes > Workflow Task Codes.

Task

Workflow task code. Click to open workflow item

Status

Workflow item status

Assigned to

User to whom the item is currently assigned.

A full history of users to whom the item has been assigned is maintainec, including the date and time they were assigned the item, and who assigned it. This can be reported on using LIFEfit's built-in report writer, or via the ODBC link in a tool such as MS Excel or Business Objects.

Start Date

Date the workflow item was created (or date used to overwrite this default)

Due date

Start date + "Maximum processing days" (from the Workflow Task Code definition) - the "red" date

Policy No

Policy number. Click to open the policy workbench.

Click here to create a new workflow task.
