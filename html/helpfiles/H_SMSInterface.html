SMS Interface Parameters

Defines the URL, user id and password to the external SMS system. The document triggers are used to call separate processes which then use the SMS interface parameters to send SMS messages.

The available SMS message can be seen on the documentation screen.

URL to SMS system

User Name

User Name supplied by the SMS provider and needed to access the SMS system

Password

Password supplied by the SMS provider and needed to access the SMS system

Text file path

File path use to hold text files that have been sent to the SMS system. Works in a similar way to the wordspool directory.

Send SMS to both Mobile numbers

Sends the SMS message to the mobile numbers listed in the Mobile and Mobile 2 field of the client record.
