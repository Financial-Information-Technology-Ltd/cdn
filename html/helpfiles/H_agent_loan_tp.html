Agent Loan Types

Insurance companies can make loans to agents. This screen determines how each type of loan will be processed.

Loans can be repaid through the agents commission statement by running batch job LF.AGENT.LOAN.REPAYMENT. It debits the agent balance (pseudo AGBAL), and credits the loan principal (pseudo AGLOAN_P) and loan interest (pseudo AGLOAN_I). The reason shown on the commission statement is LN.
