Status codes for Automatic Processing

Set the status code that will be used when a policys status changes automatically.

You are presented with a series of dropdown lists, comprising the status codes defined at Support > Codes > Status Codes of the appropriate internal status type e.g. Ready to Issue is a pending status, so you select from status codes of type P Pending.

This approach allows companies to use status codes and status definitions they are comfortable with.
