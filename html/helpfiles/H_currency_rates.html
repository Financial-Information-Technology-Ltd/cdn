Currency Rates

View and edit currency exchange rates used in transactions involving ,multiple currencies (e.g. commission on premium in policy currency USD to be paid to an agent in EUR).

In order to set or update an exchange rate, the user must enter their login ID and password. This ensures an audit trail.
