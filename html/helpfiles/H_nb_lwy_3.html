New Business - Beneficiaries and Agents

Step 3 of the new business application requires the details of the Beneficiaries and Agent. If the beneficiary is an existing client enter the client number in the client code field. If beneficiary is a new client then proceed to complete the details which will generate a new client record with a unique reference code.

Note - If No beneficiaries are specified by the policy owner, in most countries by default his/her legal heirs will be the beneficiary/ies.

Label

Description

Client Code (If exists)

Enter Client Number of Beneficiary if known which will automatically complete the beneficiary details provided LIFE fit can validate this client number.

A search option for client search is available where a client can be searched by client name, Personal ID and/or Birth date.

First Name

Enter the First Name of Beneficiary

Last Name

Enter Last Name of Beneficiary

Mobile No (if known)

Enter Mobile Number if known

Relationship

A drop down menu is displayed with a list of relationships.

Share (%)

This field is a mandatory field and requires the share of the estate to which the beneficiary is entitled. If a Policyholder has multiple beneficiaries, the total split amongst beneficiaries is 100%.

'Add a Beneficiary' Button

There may be more than one beneficiary assigned to policy owner. Once clicking the 'Add a Beneficiary' Button, further fields are displayed.

Agent Code

If the agent number is known then enter the agent number otherwise use the 'Search Agent' button to bring up a search engine. The agent can be searched by either Agent Name or Contact Name.

'Submit' Button

Press the 'Submit' button and the details entered in all the screens so far will be validated and saved. LIFE fit will allocate a unique policy number
