New Business - Policy Owner

Once selecting 'New Policy' button the new business details screen is shown. The Policy Owner details are the first details to be entered. The applicant can either be an existing policyholder for which details are already available or a New applicant which requires details on the screen to be completed. Entering Client details on this screen will generate a new client record in LIFE fit with a unique reference code.

Description

Owner Number (if already exists)

If the applicant is an existing policy holder or a previous applicant whose details have already been put into LIFE fit, enter the client number, if known.

A 'Search Client' button is available, allowing you to find a client by searching Client Name, Personal ID Number or Birth date. The more information that is provided the more refined the search will be. The results of the search will be returned in a list from which a client is selected. LIFE fit validates that this client exist and will default all the available information into the fields.
