Categories of the documents users are allowed to access from DocStore, LIFEfits inbuilt document management component

Some insurers choose to restrict access to sensitive documents. For example, they may wish certificate holders to be able to see their own medical records, but not permit them to be seen by the employer.

If left blank, all documents can be accessed.

Document Category

Select a document category from the dropdown list.

Document categories are defined in CODES record DOCUMENT_CATEGORY.

For outgoing documentation, a documents category is set as part of document setup (support > data > standard letters)

For incoming documentation, its category can be set on upload to DocStore.

It is possible to recategorise an existing document at Admin > User Adjustments > Move DocStore item this enables sensitive documents to be hidden

Sensitivity

Enter a sensitivity (1 to 10) up to which documents of this category may be accessed. 1 is the lowest and 10 the most sensitive.

For outgoing documentation, a documents sensitivity is set as part of document setup (support > data > standard letters)

For incoming documentation, its sensitivity can be set on upload to DocStore.

It is possible to recategorise an existing document at Admin > User Adjustments > Move DocStore item this enables sensitive documents to be hidden

If left blank, all documents in this category can be accessed.
