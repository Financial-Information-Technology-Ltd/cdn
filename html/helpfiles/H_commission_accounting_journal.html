Commission Adjustment

This screen enables ad hoc commission payments / clawbacks to be made for agents. It is sometimes used when bonus commission payments have been calculated "off-LIFEfit".

On submitting the screen, an account posting is made (AGBAL / XXX, where XXX is the balancing pseudo, as determined by the adjustment reason) and the agent's commission balance is updated.

There is a bulk commission adjustment process (batch job LF.IMPORT.COMMISSION.ADJUSTMENTS) which can be used as an alternative. This takes information from a CSV file and generates the appropriate accounting transactions and agent balance updates. For more information please see your system administrator, or contact FIT.
