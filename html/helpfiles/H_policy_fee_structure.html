Policy Fee Structure

Used to define a unit-linked fee charging structure which may be set at new business entry. The possible fee structures for a plan, as defined here, are set at Product Workbench > New Business.

In effect, this overrides the plan settings for policy fees and recurring commission, and avoids the necessity for a separate plan code for each possible charging structure. It is usually used for high value single premium (PPB-type) policies, where the fee structure is individually negotiated.

Label

Description

Code

Fee structure code as selected (or entered) on the previous screen display only.

Description

Free format text

Management Fee Basis

Select the management fee calculation basis from the dropdown list.

Options are:

         1 - Flat Amount

         2 - % Of Bid Value Of Fund(s)

         3 - % Of Original Investment

         4 - Premium * Comm rate

         5 - Fixed fee + % of policy value

        

Management Fee Percent

Enter the management charge percentage, if appropriate (calculation options 2, 3 and 5 above).

Management Fee Table

Select the table containing the management fee percentage from the dropdown list.

The list is comprised of rate tables defined at Support > Rates > Scheme Rates

Management Fee Amount

Enter the management fee amount, if appropriate (calculation options 1 and 5 above).

Trailer Commission only applies to base coverage

Tick if trailer commission only applies to the base coverage (i.e. no trailer commission is paid on units invested as a result of additional top-up premiums).

Trailer Comms Basis

Select the trailer commission basis from the dropdown list.

Options are:

         0 - No Trailer Commissions

         1 - Rate * Policy Value

         2 - Rate * Initial Premium

         3 - Fixed Amount

Trailer Comm Percent

Enter the fixed annual trailer commission percentage, if appropriate.

Trailer Comm Table

Select the rate table containing the annual trailer commission percentage, if appropriate, from the dropdown list.

The list is comprised of rate tables defined at Support > Rates > Scheme Rates

Trailer Comm Amount

Enter the fixed trailer commission amount, if appropriate.

Duration

Duration in years for which trailer commission is paid. After this period, no UTC future records will be generated.

If blank, trailer commission will be paid until the policy ends.

<Add> button

Add another duration row to the trailer commission table, so trailer commission can change over time.

Renewal Comms Basis

Select the renewal commission basis from the dropdown list.

Options are:

         1 - Rate * Policy Value

         2 - Rate * Initial Premium

Renewal Comm Percent

Enter the renewal commission percentage, if appropriate.

Renewal Comm Table

Select the rate table containing the renewal percentage, if appropriate, from the dropdown list.

The list is comprised of rate tables defined at Support > Rates > Scheme Rates.

Duration

Duration in years for which trailer commission is paid.

If blank, renewal commission will be paid until the policy ends.

<Add> button

Add another duration row to the renewal commission table, so the renewal commission can change over time.

Admin Fee Basis

Select the administration fee calculation basis from the dropdown list.

Options are:

         1 - Flat Amount

         2 - % Of Bid Value Of Fund(s)

         3 - % Of Original Investment

         4 - Premium * Comm rate

         5 - Fixed fee + % of policy value

Admin Fee Percent

Enter the administration fee percentage, if appropriate (calculation options 2, 3 and 5 above).

Admin Rate Table

Select the rate table containing the administration fee percentage, if appropriate, from the dropdown list.

The list is comprised of rate tables defined at Support > Rates > Scheme Rates.

Admin Fee Amount

Enter the fixed trailer administration fee amount, if appropriate.

Admin Fee Duration

Duration in years for which administration fee is charged.

If blank, the administration fee will be charged until the policy ends.

<Add> button

Add another duration row to the administration fee table, so the administration fee can vary over time.

Amortise Policy fee

Tick if the policy fee is to be amortised.

Amortise Initial Fee

Tick if the initial fee is to be amortised.

Amortise Mort Charge

Tick if the mortality charge is to be amortised.
