Batch Control

Used to open and closer a batch of payments, to confirm that the amount of cash entered as payments matches the total on the bank statement.

Label

Description

Operator id

Display only.

Processing date

Current system date display only.

Batch number.

If opening a batch, the next available number for a batch. If closing a batch, the current batch number.

Cash

Enter the amount of cash from the bank statement. On closing a batch, this is verified against the total amount of cash entered on LIFEfit sine the batch was opened.

Cash entered

Display only - the total amount of cash entered on LIFEfit sine the batch was opened.

<Open batch> /

<Close batch> button

If the batch is currently closed, the <Open batch> button is shown.

If a batch is currently open, the <Close batch> button is shown.

<Return> hyperlink

Return to the main menu.
