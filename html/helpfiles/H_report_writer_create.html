Report Definition

Select an existing report to edit, or create a new one. These are sometimes referred to as "Davywriter" reports.

Label

Description

Select Report Name

If you know the report name, enter it. If it does not exist, a new report will be created.

The drop down shows the reports to which you have access - you either created them, or you are in a user group which is allowed to edit them.

The order of the drop down is:

-           first, reports you created are listed alphabetically

-           then, those which you can also access by virtue of your user group are listed, also alphabetically

"Define" button

Takes you to the screen to create / edit the report
