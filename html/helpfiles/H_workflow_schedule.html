Workflow Schedule

List of scheduled workflow items.

This screen lists regular, non-policy specific tasks for which workflow reminders and tracking are useful e.g. monthly accounts processing, quarterly distribution processing, annual reminder to set public holidays in LIFEfit.

Heading

Description

Active

Is this scheduled task active (i.e. do you want regular workflow items created) - yes or no.

Task

Workflow task type

Next Date

Next scheduled date for this task.

Batch job LF.WORKFLOW.SCHEDULE checks whether each active item's "next date" is within five days of the current server date (n.b. not LIFEfit system date), and creates a workflow item if necessary. It will be visible from the due date (even though it may have been created up to five days in advance).

The item will appear in the user group "to do" list for the members of the default user group, as defined on the workflow task code (Support > Codes > Workflow Task Codes).

Frequency

Frequency of the scheduled task - weekly, monthly, quarterly or annual.

Start Date

Date item was first scheduled for.

End Date

End date - after which the task will no longer be scheduled.

If blank, the task will be scheduled indefinitely.

Description

Free format description, as entered when the schedule was created.

"Create New Item" button

Takes you to the screen to create a new scheduled task.

"Show Inactive Items" button

"Hide Inactive Items" button

Enables you to just include active items in the list, or all items.
