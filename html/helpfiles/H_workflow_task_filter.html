Workflow Task List

This screen is used to provide simple reports of workflow items on LIFEfit.

It can be used by managers and supervisors to give a high level overview of the outstanding work situation in servicing departments. It is also useful if work needs to be reassigned due to bottlenecks or absence.

Label

Description

Assigned to user

Restricts the list by user ID to whom tasks are assigned - select from the drop down list.

Status

Restricts the list by task status - select from the drop down list.

Task Code

Restricts the list by workflow task code - select from the drop down list.

Due before or on

Restrict the list by due date (based on maximum allowed processing time for the task)

Policy no

Only list workflow items for a specific policy number

"Continue" button

The list is displayed, including the RAG (Red Amber Green) Level.

Click on the "Task" hyperlink to see details of the workflow item.

Click on the Policy number hyperlink to open the policy workbench screen.

"New Task" button

Creates a new workflow item - takes you to the "new workflow" screen (Workflow > New Task)
