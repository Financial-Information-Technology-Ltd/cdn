Agent Loan

This screen allows an approved Agent Loan to be keyed onto the system. A Payment out will be generated for the loan advance.

Label

Description

Loan Amount

The loan amount.

Start Date

The effective date of the loan advance.

Loan Term (Months)

The Term in Months for the loan to be repaid. This will be used to calculate the modal repayment amount.

Loan Type

Select the loan type for this loan. The loan type has settings which determine how the interest is calculated and repayment details. Loan Types are defined at Support Files > Codes > Agent Loan Types.

Payments Out heading

% (display only)

The loan advance can only be paid to one payee. The payee defaults to the agent.

Type

The method of payment. This is a drop down list based on the available methods defined at Admin > Configuration > System Date Limits / General, "Payment Out Types ". This field defaults to the agents usual payment method for commission.

Charge amount

Amount to deduct for bank charges. Will be posted to accounting pseudo BANK_CHARGE

Payee No

LIFEfit client number of the payee. Defaults to the Agent.

Bank no

The bank account to which the payment is being made - only required for electronic payments.

Edit

Allows you to amend an existing bank account record to be used for the payment (only required for electronic payments)

Create

Allows you to create a new bank account record to be used for the payment (only required for electronic payments)

Additional instructions

Free format text. Will be used in the electronic payment file (TT or direct credit)
