Unit Statement Print at Policy Anniversary

If checked, the system will produce annual Premium Statements for all policies. The statement for a policy will be generated by the processing its AP future record.

The Unit Statement process must be set for each plan (on the plans valuation screen). There is no associated document definition.

The template name must be in the format required by the unit statement process program - for example: PremiumStatement{PlanCode]_{LanguageCode}.

Agent Persistency %

An agents persistency is a measure of the quality of business he/she writes. Agents who write poor quality business (with a high proportion of lapses) are penalised with higher levels of commission clawback.

This figure represents the threshold for deciding how much commission to claw back on policy lapse.

If an agent has a persistency figure above this, he/she can keep any commission earned after the number of months below (i.e. only the unearned commission amount remaining is clawed back).

If an agent has a persistency figure below this, and any commission remains unearned, all commission paid is clawed back

Numeric, two decimal places.

Agent Persistency Months

Used in conjunction with Agent Persistency % above.

If a policy lapses with a duration (issue date to paid to date) less than this figure, all commission paid is clawed back, irrespective of the agents persistency figure.

If a policy lapses on or after this duration, and an unearned commission balance remains, the amount of clawback will be determined by the agents persistency figure.

Integer

e.g.

1.      Agents persistency is 75%, Agent Persistency % above is 85%, Agent Persistency Months is 6.

A policy issued on 01/01/2017 lapses with paid-to date 01/08/2017 (7 months duration) and the unearned commission balance is 500.00. Original commission amount paid was 1,200. 1,200 is clawed back as the agents persistency is below 85%

2.      Agents persistency is 87%, Agent Persistency % above is 85%, Agent Persistency Months is 6.

A policy issued on 01/01/2017 lapses with paid-to date 01/08/2017 (7 months duration) and the unearned commission balance is 500.00. Original commission amount paid was 1,200. 500.00 is clawed back as the agents persistency is above 85%
