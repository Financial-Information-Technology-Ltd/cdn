The Bank Identifier Code is a standard format code (also known as SWIFT-BIC, BIC code, SWIFT ID or SWIFT code) approved by the International Organization for Standardization (ISO). BIC codes are used when transferring money between banks, particularly for international wire transfers, and also for the exchange of other messages between banks.
