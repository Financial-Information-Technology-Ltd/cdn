Audit Logs

LIFEfit maintains a log of changes to its parameter files as selected at Administration > Configuration > Audit logging. It can be useful to see what changes have been made, when they were made and who made them.

Label

Description

Tables

Select a table you wish to report on.

Key Value (optional)

If you wish to report changes to a specific record on the table, enter its key. This is often used to report on change to a single plan code.

If you leave this blank, all changes to all records will be reported.

<Submit> button

Generates a CSV file of changes to the selected table/record(s)

-         record key

-         date

-         time

-         field

-         field value before

-         field value after

The csv file can be loaded from hyperlink or, if too large (> 5Mb), can be opened directly from the &hold& folder.

<Return> hyperlink

Return to the main menu.
