Entry Of Diary Details

This screen is used to create policy diary reminders, which are automatically emailed to users. It creates DY future activities for one or more policies, which trigger emails to the user.

The batch program used to process this future record is LFBDR, this looks forward a given number of days which is set in Administration>Configuration>System Date Limits/General, Diary Days and sends an email as a reminder.

Label

Description

Date

The date the future record will be created for.

Details

Free-format text of the diary reminder.

Something must be entered here as it cannot be left blank.

Policy Numbers

These are the policy numbers that will have a future record created. One email is sent for each.

User

The user to whom the email will be sent. The user can either be selected from the list by clicking on the search button or they can be entered manually. The email address is set on the user profile (if there is no email address, the future record will not be processed).

Add button

Clicked to add more polices.

Save button

This saves the information and creates DY future records.
