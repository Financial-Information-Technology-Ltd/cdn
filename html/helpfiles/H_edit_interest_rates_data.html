Interest Rates

This screen is used to set loan interest rates which can vary according to loan amount and loan term

Label

Description

Rates Code

Rates Date

Date from which the rate table is active

Maximum Amount

Column headings loans up to this amount will be charge an interest rate from this column

Max. Month

Row headings loans up to this duration will be charged an interest rate from this row
