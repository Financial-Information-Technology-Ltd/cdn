General Ledger Pseudo Codes

Each financial transaction in LIFEfit results in an accounting journal entry to a pre-determined set of accounting pseudo codes. Pseudo codes are mapped to account numbers at company, plan or fund level as appropriate, to deliver general ledger information according to the companys chart of accounts.

Set the attributes of the pseudo code on this screen.

Label

Description

Pseudo Code

Display only as selected (or entered) on the previous screen

Description

Pseudo description display only

Account Type

Select the account type from the dropdown list.

Options are:

         A - Assets

         B - Balance Sheet

         C - Claim

         E - Expense

         I - Income

         L - Liability

         M - Memo

         P - Profit & Loss

         T - Tax

This determines how accounts are group on the trial balance report (LF013).

Map Location

Select the account mapping location from the dropdown list, if permitted.

Options are:

         F - On Funds

         N - Mapping Not Allowed (account is mapped on company)

         P - On Plans

Pseudo groups

Select the groups to which the pseudo belongs, if any, from the dropdown list.

Options are:

         C - Cash Book Accounts. Will be offered in the cash pseudo dropdown when posting "cash", if multiple cash books are used.

         S - Suspense Accounts

         P - Restricted Policy Accounting. Will be unavailable on the dropdown of account pseudos to be used as balancing entries on the policy accounting screen.

         MCR - Margin Control Report. Calling batch job LF.ACCOUNTING.GROUP.REPORT with parameter "MCR" will include this account.

         PO - Payments Out Cash Pseudos. Will be available from the dropdown when setting the payment out cash pseudo to be used for a payment out type (e.g. cash, direct credit)

         CA - Claim Adjustments Accounts. Will be available from the dropdown when authorising a claim payment.

n.b. this list comes from CODES record PSEUDO_GROUPS and can be extended (if, for example, there are sets of pseudos you want to report on in a similar way to "MCR" pseudos as described above).

<Add>

Add another pseudo grouping to the list.

Ledger Cd

Select whether the account appears on the company ledger or fund ledger (for nominal leger export e.g. to Sun Accounts some companies choose to only take company accounts into Sun, and leave detailed Fund level analysis in LIFEfit)

<Save> button

Save changes.

<Return>

Return to the previous screen without saving changes (if any).
