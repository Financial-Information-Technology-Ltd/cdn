Agent Balance transfer

This screen enables a commission balance to be moved from one agent to another. The source agent's account is debited, and the target agent's account is credited. The necessary accounting transaction is generated.

Label

Description

From Agent

Agent code of the "source" agent - the agent from whom commission is being taken.

This is validated, and the agent name is displayed.

"Search Agent" button

Enables the user to search for agent by name or contact name.

To Agent

Agent code of the "target" agent - the agent to whom commission is being given.

This is validated, and the agent name is displayed.

Currency

Currency of the balance transfer

Amount

Amount of the transfer.

n.b. this is NOT limited by the current credit balance of the "source" agent - it can be more, leaving the "source" agent with a negative balance.

Branch

Select from the drop down list of branches.

Description

Free-format text.

Something must be entered here - it cannot be left blank.

"Submit" button

Creates a transaction in the general accounting journal file (GENACCTRN).

This will be posted to the general ledger and copied to journal history file GENACCTRN_POSTED the next time the accounting batch is run.
