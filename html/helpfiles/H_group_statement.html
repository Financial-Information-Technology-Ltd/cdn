Statement Type

Identifies the version of the statement to be produced. Options are:

  • Short Statement This statement shows scheme totals only. There is a total for each date and transaction type.
  • Long Statement This shows the transactions for each member. For GDA policies, amounts are shown as Employer and Employee.
  • One Line Per Member This statement shows the total for each transaction type for each member.
