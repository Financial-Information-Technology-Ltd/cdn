Bill Type

Define the attributes of the bill type selected (or entered) on the previous screen. The bill types for a plan, one for each of its mode method combinations, are selected from the bill types defined here. They are set on the Product Workbench > Billing screen.

Description

Code

Name

Event Name

Document event trigger can be linked to a specific merge document either at Admin > Config > Documentation, Event (if common for all products), Or Product Workbench > Documentation (if plan-specific).

Billing Notice (days before due day)

Number of days before the premium due date (paid-to date) that the bill should be triggered. This is the date at which a BI future record will be created.

Available for Group Business
